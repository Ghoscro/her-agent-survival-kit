# Checkpoint Template

Use this before context compaction, model switch, long pause, or handoff.

```text
Goal:
Why it matters:
Done:
Evidence:
Not done:
Open risks:
Next smallest action:
Do not touch:
```

## Completion test

A checkpoint is good if a fresh agent can answer:

1. What are we trying to finish?
2. What changed already?
3. What proves it?
4. What is unsafe or excluded?
5. What should happen next?
