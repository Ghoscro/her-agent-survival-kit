# Security and Public Boundary

This repository is a public-safe starter kit.

Examples use mock data. Private memory, user data, credentials, deployment details, protected persona/source files, and full automation loops are intentionally excluded.

## Do not contribute

Please do not open issues or pull requests containing:

- API keys, tokens, cookies, passwords, SSH keys, or `.env` files
- real user data or private chats
- private memory dumps
- private persona/soul files
- production hostnames, internal paths, or deployment scripts
- invite-code pools or access-control details

## Reporting

If you believe sensitive data was accidentally included, please open a minimal issue that names the file and line without repeating the sensitive content.
